Phase One - Create The Project

Step Name Description
Step One Open VS Code Open whatever Text-Editor you are using on your computer
Step Two Create A Folder Create a new folder for the project you are working on
Step Three Create An HTML File Create a new file titled index.html
Step Four Create A CSS File Create a new CSS file called styles.css
Step Five Link the CSS file to the HTML file In the head section of the HTML file, type the link element and fill in the rel and href items

Phase Two - Basic HTML Elements

Element Name Description
<head> An element that contains info about the page but won't be seen on the webpage
<title> An element within the head element that will be the name at the top of the browser or tab
<body> An element that contains the parts of the webpage that will be visible
<h1> A header with the largest size font
<h2> A header with the second largest size font
<h3> A header with the third largest size font
<h4> A header with the fourth largest size font
<h5> A header with the fifth largest size font
<h6> A header with the smallest size font
<p> Text that would be used in a paragraph in small font

Phase Three - HTML Text Modifier Elements

Element Name Description
<b> The characters become bold
<i> The characters become italicized
<sup> The characters become a superscript like writing th for a date
<sub> The characters become a subscript like writing a chemical formula
<br /> Creates a break where the words begin a new line
<hr /> Creates a horizontal line seperating two lines of text
<strong> Changes the characters to bold and creates emphasis for screen readers
<em> Changes the characters to italics and creates emphasis for screen readers
<blockquote> An element for long quotes that will indent the quote and place it on a new line
<q> Used for shorter quotes and adds quotation marks except for on Explorer
<s> Creates a strikethrough on the text

Phase Four - HTML List Elements

Element Name Description
<ol> Creates an ordered list with numbers
<ul> Creates a list with bullet points
<li> Creates a new list item

Phase Five - HTML Links, Pictures and Videos

Element Name Description
<a> Creates a link with an href attribute for the source of the link which could be absolute (on the web) or relative (on the computer)
<img> Allows you to add an image with the source and an alt attribute which provides a text description of the image if it cannot be viewed
<figure> Used as a parent tag to the img tag to allow for a caption using the figcaption tag
<figcaption> Used to write a caption under an image
<video> Allows you to insert the source of a video; you include controls or autoplay to determine the format of play
<audio> Allows you to insert the source of an audio file; you can also add controls or autoplay

Phase Six - Tables in HTML

Element Name Description
<table> Creates a new table
<thead> Creates the headings of the table
<th> Allows you to input the text titles of the headings
<tbody> Creates the body of the table
<tr> Creates a new row in the table
<td> Allows you to add text or data to the table; you can add colspan or rowspan to have the info spread across columns or rows
<tfoot> Allows you to add a footer in the table

Phase Seven - Additional Markups in HTML

Element Name Description
<!-- --> Creates a comment that will not appear on the webpage
<p class='blah'> Creates a class named blah that can be modified in the CSS code
<p id = 'blahblah'> Creates a more specific tag named blahblah called an id that can be modified in CSS code
<div> Creates a group of elements that can be modified in CSS
<span> Similar to the inline version of the div element
